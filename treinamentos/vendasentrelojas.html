Treinamento: Vendas entre Lojas e Transferências de Mercadoria

Regra Principal

A operação a ser realizada depende da Razão Social:

  • Transferência: Circulação entre lojas da mesma Razão Social (ex: Marqui Venturini entre si).
  • Venda: Circulação entre lojas de Razões Sociais diferentes (ex: Marqui Venturini para Tamandaré Tintas)

Transferências

Para transferências, devemos utilizar a rotina de Outras notas localizada em VENDAS → NOTA FISCAL → OUTRAS NOTAS.

Dados importantes:
Vendedor: 1099 -- Vendas entre lojas do grupo.
Natureza: 009 -- Transferência entre lojas.
Preços: 4 -- Custo Médio.
⚠️ ATENÇÃO - Preço Zerado: Se o produto aparecer com preço R$ 0,00, a loja deve pedir para a matriz consultar o Preço de Custo Médio e inseri-lo manualmente na NF.
  • Loja Morada: Matriz das lojas Tamandaré Tintas.
  • Loja Visconde: Matriz das lojas Marqui Venturini.
💡 Dica: É possível aproveitar o pedido feito pelos vendedores para não precisar digitá-lo novamente. Use o atalho F8 para abrir a janela onde é possível informar o número do pedido a ser carregado.

Vendas entre lojas

Nas vendas, o processo é um pouco diferente.
Esse tipo de operação deve ser realizado em VENDAS → LANÇAMENTO DE VENDAS, pois se trata, de fato, de uma venda entre lojas.

O procedimento é praticamente igual a uma venda comum, com apenas duas diferenças importantes:

  1. Forma de pagamento: deve ser sempre CA.
  2. Preço: deve ser aplicado o desconto diferenciado definido para vendas entre lojas.

Desconto para Vendas entre lojas

Para venda entre lojas temos duas formas de desconto a depender do tipo de produto que vamos vender.

Tintas de máquina

Em caso de tintas de máquina — seja Suvinil, Advance, Lazzuril, Alessi ou outras — o desconto é sempre o mesmo:
➡️ 30% de desconto aplicado no total da NF, na aba "Desconto".

Importante:
Neste tipo de venda devem constar apenas produtos da máquina.
Não deve haver nenhum outro produto incluído, tais como catalisadores (Parte B ou C), thinners ou qualquer outro item.

Se, ao selecionar o produto na máquina, o sistema adicionar automaticamente a Parte B, ela deve ser excluída do pedido, e uma nova venda separada deve ser realizada exclusivamente para esse item.

Demais produtos

Para produtos como Parte B, Parte C, Thinners e todos os demais que não são tintas de máquina, o desconto segue um processo diferente:
➡️ Devem ser vendidos a preço de custo.

Dados Importantes
Vendedor: 1099 — Vendas entre lojas do grupo
Forma: CA

Passo a Passo para Localizar e Aplicar o Custo Médio:

  1. Clique sobre o produto na venda.
  2. Use o atalho F7 (Função "Margem").
  3. Clique na Lupa (Detalhes).
  4. Clique no ícone % (Negociar).
  5. Clique no ícone Positivo (Autorizar).
  6. Na tela, substitua o "Preço negociado" pelo valor do Custo Médio.
