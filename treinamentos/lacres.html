O que é o novo sistema e por que estamos usando?

Este sistema é digital e serve para registrar a numeração de todos os lacres dos malotes. [cite: 3] Ele garante o registro correto de cada lacre, sem erros de digitação, com data, hora e nome do operador. [cite: 4]

Benefícios para você:

  • **Segurança no processo:** O sistema garante que os números dos lacres são registrados de forma precisa. [cite: 6]
  • **Agilidade na conferência:** As informações chegam na mesma hora para a equipe de conferência, agilizando a checagem do malote. [cite: 7]
  • **Transparência e proteção:** Todas as informações são rastreadas, o que protege você e a loja em caso de divergência. [cite: 8]

Sua Responsabilidade

É sua responsabilidade garantir que os lacres sejam registrados corretamente antes de serem enviados. [cite: 10]

Passo a Passo para o Registro

1. Acesse o site: Use o link fornecido pela liderança para abrir o site no navegador (celular ou computador). [cite: 12]
2. Digite o número do lacre: Digite a numeração no campo "Número do Lacre". [cite: 14]
**ATENÇÃO:** O sistema **não permite copiar e colar** para garantir que a digitação seja manual e sem erros. [cite: 15, 16]
3. Confirme a numeração: Digite o mesmo número no campo "Confirmar Lacre". [cite: 18]

Se os números não forem iguais, a tela mostrará a mensagem de alerta ("Atenção: Os números de lacre não são iguais") [cite: 19] e o campo ficará com uma borda vermelha para que você possa corrigir. [cite: 20]

4. Preencha os dados:
  • **Loja:** Selecione o nome da sua loja na lista. [cite: 22]
  • **Operador:** Digite o seu nome. [cite: 23]
5. Finalize o registro: Clique no botão **"Registrar Lacre"**. [cite: 24]

Mensagens do Sistema

Lacre registrado com sucesso!: O formulário de registro sumirá e essa mensagem aparecerá, indicando que o lacre já foi registrado. [cite: 26, 30]
Erro ao enviar os dados: Houve um problema de conexão. Por favor, verifique sua internet e tente novamente. [cite: 27, 28]

Próximo Passo

Após a mensagem de sucesso, clique no botão **"Novo Registro"** para registrar o próximo lacre. [cite: 31]
