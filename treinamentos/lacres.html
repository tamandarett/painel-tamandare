O que é o novo sistema e por que estamos usando?

Este sistema é digital e serve para registrar a numeração de todos os lacres dos malotes. Ele garante o registro correto de cada lacre, sem erros de digitação, com data, hora e nome do operador.

Benefícios para você:

  • Segurança no processo: O sistema garante que os números dos lacres são registrados de forma precisa.
  • Agilidade na conferência: As informações chegam na mesma hora para a equipe de conferência, agilizando a checagem do malote.
  • Transparência e proteção: Todas as informações são rastreadas, o que protege você e a loja em caso de divergência.

Sua Responsabilidade

É sua responsabilidade garantir que os lacres sejam registrados corretamente antes de serem enviados.

Passo a Passo para o Registro

1. Acesse o site: Use o link fornecido pela liderança para abrir o site no navegador (celular ou computador).
2. Digite o número do lacre: Digite a numeração no campo "Número do Lacre".
ATENÇÃO: O sistema não permite copiar e colar para garantir que a digitação seja manual e sem erros.
3. Confirme a numeração: Digite o mesmo número no campo "Confirmar Lacre".

Se os números não forem iguais, a tela mostrará a mensagem de alerta ("Atenção: Os números de lacre não são iguais") e o campo ficará com uma borda vermelha para que você possa corrigir.

4. Preencha os dados:
  • Loja: Selecione o nome da sua loja na lista.
  • Operador: Digite o seu nome.
5. Finalize o registro: Clique no botão "Registrar Lacre".

Mensagens do Sistema

Lacre registrado com sucesso!: Se tudo estiver correto, o formulário de registro vai sumir e essa mensagem aparecerá na tela. O lacre já está registrado.
Erro ao enviar os dados: Se essa mensagem aparecer, houve um problema de conexão. Por favor, verifique sua internet e tente novamente.

Próximo Passo

Após a mensagem de sucesso, você pode clicar no botão "Novo Registro" para registrar o próximo lacre.
